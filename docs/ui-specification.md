## 3. 各パネル詳細仕様

### 3.1 [A] トップバー

```
┌────────────────────────────────────────────────────────────────────────┐
│ ◆ NovaPlex Core  │ 67.42 SOL          │ +12.38 SOL (+24.8%)         │
│                  │ ≈ $9,978            │ Today +1.84 (+2.9%)         │
│                  │                    │ W/R 67.3%  ● 3 Bots Active  │
└────────────────────────────────────────────────────────────────────────┘
```

**構成要素 — 左から右へ 3 カラム構成:**

**① ロゴカラム (左端, 固定幅)**
- アプリ名 `NovaPlex Core` + 紫アクセントの六角形アイコン
- フォント: Inter 600 / 15px — 控えめだが読みやすいサイズ

**② 残高カラム (中央)**
- **SOL 残高**: JetBrains Mono 600 / **20px** — トップバー内で最大フォント
  - 数値のみ大きく、通貨ラベル `SOL` は 13px で右寄せ
- **USDC 換算**: text-secondary / 12px で下行に控えめ表示
  - 「≈ $9,978」のように概算表記で "桁を見せる" が主張しすぎない

**③ PnL カラム (右側) — 「自然に目を引く」デザイン**

| 要素 | フォント | 色 | 演出 |
|------|---------|-----|------|
| 全期間 PnL 数値 | JetBrains Mono 600 / **18px** | accent-green (利益時) / accent-red (損失時) | — |
| 全期間 PnL % | JetBrains Mono 400 / 13px | 同上だが **opacity: 0.75** | 括弧付き |
| Today PnL | Inter 400 / 12px | text-secondary ベース + 微かな緑/赤 | — |
| 勝率 | Inter 500 / 12px | text-secondary | 数値のみ、装飾なし |
| Bot ステータス | Inter 400 / 12px | text-muted + 緑パルスドット | — |

**PnL を「嘘くさくなく目立たせる」ための設計原則:**

1. **フォントサイズ差で視線誘導**
   - 残高 20px > PnL 18px > 他 12px の 3 段階グラデーション
   - PnL を残高より 1 段小さくすることで「見つかる」が「押し付けない」
2. **色は accent-green だが彩度を抑制**
   - PnL 数値: `accent-green` そのまま (#0ecb81)
   - PnL %: `opacity: 0.75` で少し落として二重強調を避ける
   - Today PnL: `text-secondary` に accent-green を **20% ミックス** した中間色
3. **背景ハイライトは使わない**
   - 背景色・グロー・ボーダーなどの追加装飾は一切なし
   - 周囲の暗い bg-secondary に対して文字色だけで自然に浮く
4. **変動時アニメーションは控えめ**
   - 数値変化: `AnimatedNumber` で 500ms かけて滑らかにカウント (急にジャンプしない)
   - 色フラッシュ: **なし** (トップバーはフラッシュさせない — 安定感を優先)
   - 代わりに: PnL 横に極小の `▲` / `▼` 三角アイコン (4px) で方向だけ示す
5. **パーセンテージは控えめな位置**
   - `+12.38 SOL` が主役、`(+24.8%)` は同行右に小さめで併記
   - % が大きすぎると胡散臭くなるため、SOL 実数を先に見せる

## 4. アニメーション仕様

### 4.1 価格更新アニメーション

| 要素 | アニメーション | Duration | Easing | 備考 |
|------|--------------|----------|--------|------|
| 価格変動 (上昇) | テキストが一瞬緑にフラッシュ | 300ms | ease-out | ウォッチリスト・ポジション内のみ |
| 価格変動 (下落) | テキストが一瞬赤にフラッシュ | 300ms | ease-out | ウォッチリスト・ポジション内のみ |
| トップバー PnL | AnimatedNumber (滑らかカウント) | 500ms | ease-in-out | **色フラッシュなし** — 安定感優先 |
| トップバー残高 | AnimatedNumber (滑らかカウント) | 500ms | ease-in-out | **色フラッシュなし** |
| 新規トレード追加 | 上からスライドイン + フェードイン | 400ms | ease-out | |
| ポジション更新 | 行の背景が微小にパルス | 200ms | linear | |

### 4.2 ログアニメーション

- 新しい行: 左からタイプライター風に表示 (1 文字 10ms)
- 自動スクロール: smooth scroll behavior
- カーソル: 0.5 秒間隔でブリンク

### 4.3 チャートアニメーション

- 新しいローソク足: 右からスライドイン
- ローソク足更新: Close 位置がリアルタイムに伸縮
- Bot マーカー: ポップイン + パルスエフェクト
